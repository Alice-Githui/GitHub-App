### Application Name
Github-App

This project was created by Alice Githui.

## Description
This website allows the user to enter their GitHub username, after which they can retrieve the names and descriptions of their public repositories. It also allows the user to search within their repositories. The application makes use of GitHub API. 

## Technologies Used

-Angular/TypeScript
-HTML 
-CSS

## Installation and Set-up

To acquire a local copy:

- Clone the repository
- Navigate to the parent folder
- View the app from the root/app component

# GitHub App - Build and Deployment

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Code scaffolding
